API documentation

ElecSal

Methods GET
Usuarios

https://localhost:8080/usuario/findAll Obtiene todos los elementos de la tabla

https://localhost:8080/usuario/findById/{id} Obtiene un elemento por id

Partidos

https://localhost:8080/partidos/findAll Obtiene todos los elementos de la tabla

https://localhost:8080/partidos/findById/{id} Obtiene un elemento por id

Cargos

https://localhost:8080/cargos/findAll Lista de cargos existentes

https://localhost:8080/cargos/findById/{idCargo} Lista de cargos existentes

VotosCandidato

https://localhost:8080/votosCandidatos/findAll Obtiene todos los elementos de la tabla

https://localhost:8080/votosCandidatos/findById/{id} Obtiene un elemento por id

https://localhost:8080/votosCandidatos/findDiputadosDepartamento/{idDepartamento} Lista de candidatos a diputados por Departamento

https://localhost:8080/votosCandidatos/findAlcaldesMunicipio/{idMunicipio} Lista de candidatos a Alcaldes por municipio

https://localhost:8080/votosCandidatos/findPresidentes Lista de candidatos a Presidentes

https://localhost:8080/votosCandidatos/diputadosElegidos/{idDepartamento} Lista de diputados elegidos por departamento

https://localhost:8080/votosCandidatos/alcaldeElectoMunicipio/{idMunicipio} Alcalde electo

https://localhost:8080/votosCandidatos/presidenteElecto Presidente electo

Method POST
Usuario

https://localhost:8080/usuario/login Recibe un objeto de tipo Usuario, puede enviar solamento dui y password

{ "dui": 222222222, "contrasenia": "98765" }

VotosCandidatos

https://localhost:8080/votosCandidatos/crearVotoscandidatos Recibe un objeto de VotosCandidato

{
"idUsuario": {"idUsuario":8 },
"votos": 0,
"idCargo": {
"idCargo": 1
},
"idPartido": {
"idPartido": 1
}
}

Method PUT
Usuario

https://localhost:8080/usuario/checkvoto Recibe el objeto Usuario con el id y estadoVoto editado

{ "idUsuario":{"idUsuario":9} ,"estadoVoto": true }

VotosCandidato

https://localhost:8080/usuario/checkvoto Recibe el objeto Usuario con el id y votos editado

{ "idUsuario":{"idUsuario":100} ,"votos": 1 }
